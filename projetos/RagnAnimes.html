RagnaAnimes Front End

RagnAnimes Plataforma com front-end desenvolvido em HTML, CSS e JavaScript para explorar animes por categorias. Inclui cards interativos com imagens, títulos e descrições, filtro por categorias e interface responsiva, demonstrando habilidades em design, manipulação do DOM e desenvolvimento web moderno. Aviso: Este projeto não foi desenvolvido de forma individual, mas sim em colaboração com uma equipe.
